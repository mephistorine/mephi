# Что это?

Разберем это на примере компонента из [Angular Material](https://material.angular.io/) — [Chips](https://material.angular.io/components/chips/overview).

![Material Chips](/articles/what-is-angular-module/material-chips.png)

Использовать его, достаточно просто, в ближайшем модуле нужно подключить модуль `MatChipsModule`. Затем можно уже начинать использовать сами компоненты:

```html
<mat-chip-list aria-label="Fish selection">
  <mat-chip>One fish</mat-chip>
  <mat-chip>Two fish</mat-chip>
  <mat-chip color="primary" selected>Primary fish</mat-chip>
  <mat-chip color="accent" selected>Accent fish</mat-chip>
</mat-chip-list>
```

Как можно увидеть тут используются 2 компонента это `mat-chip-list`, `mat-chip`. Интересная вещь в том, что мы просто подключили один модуль и нам доступно два компонента, но на самом деле больше. То что доступно после того как вы подключили модуль зовется — **публичным API**.

![](/articles/what-is-angular-module/private-api.png)

❗ Модуль — сущность, которая умеет объединять остальные сущности Angular под один так сказать зонтик, а на внешнее использование предоставлять только то что посчитает нужным.

В рамках самого модуля может происходить что угодно, там может быть большое количество компонентов и сервисов которые взаимодействуют друг с другом, но во внешнее API может только маленькая часть.

К примеру есть условный MatChipService(в реальности ничего подобного нет), он что-то хранит и создается в рамках MatChipsModule. Таким образом его используют только в рамках этого модуля по какой-то причине, во внешнем API он нам не нужен.

![](/articles/what-is-angular-module/private-api-2.png)

Как раз для этого и нужны модули.

## Пример из жизни

Самый яркий пример это автомобиль. Вы примерно знаете как работает машина изнутри, но не досконально, это не плохо, это показывает что вам необязательно знать об этом, чтобы пользоваться ею то о чем мы говорим как раз таки есть — **внутренне API**.

![](/articles/what-is-angular-module/inside-in-car.png)

Все что есть у вас, как у пользователя этой машины, это руль, педали, коробка передач, ручник и всякое мелкое на панели управления все это зовется — **публичным API**.
